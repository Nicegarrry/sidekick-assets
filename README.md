# sidekick-assets

Static asset hosting for the sidekick Notion projections.
Images are served via raw.githubusercontent.com and used as Notion page covers and image blocks.

## Layout
- `wine-labels/` — resized wine label photos (JPEG, ≤400 KB each)
- `charts/` — generated matplotlib chart PNGs (PNG, ≤500 KB each)
- `banners/` — Atlas-style page/DB cover banners

Managed by the sidekick `notion-visuals` skill. Do not hand-edit.
